How to play game

This is a game that has two level easy and hard when you click on one of the button a svg animation will appear which Shows Start game and game will start. The game is that we have a snow man which user can move to right or left by moving mouse or by keyborad and should catch snow balls. Each snow balls that it gets the score will increase. we have a score counter that when game finished it will stop. We also have a selector that user can choose the color of the snowballs. This snowman will catch the snowballs only by head and so try to catch them before that they pass.
